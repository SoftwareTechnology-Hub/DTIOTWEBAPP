How to Use Your API Key

1
Copy Your API Key

Click the "Copy" button above to copy your unique API key to clipboard.

2
Include in API Requests

Add your API key in the request header:

Authorization: Bearer YOUR_API_KEY
3
Example Request
curl -X GET https://api.yourapp.com/data \ -H "Authorization: Bearer YOUR_API_KEY"

Troubleshooting

API Key Not Working?

Make sure you're using the correct format in the Authorization header and that your API key hasn't expired. Try regenerating your key if issues persist.

Getting 401 Unauthorized Error?

This usually means your API key is missing or invalid. Double-check that you've copied the entire key and included it in the Authorization header.

Rate Limit Exceeded?

You've made too many requests in a short time. Wait a few minutes before trying again or upgrade your plan for higher limits.
